# chatgpt-wechat-mp
##无需编程即可将chatgpt接入自己的微信公众号

第一步：配置公众号白名单
假设你已经注册好了一个公众号。在下面的基本配置中，点击启用“开发者密码(AppSecret)”。配置完之后，会在下面产生一个IP白名单，进入白名单后进行添加。需要添加的IP白名单请在乐乐感知学堂公众号内回复“IP白名单”即可获取。
![image](https://user-images.githubusercontent.com/27985664/209424517-0646f3e2-ed90-407f-b391-e6df6eb0cef7.png)
第二步：配置服务器
  仍然是在图中的基本配置中进行配置，下方的“服务器配置”。
  （1）点击右侧“修改配置”，添加服务器地址和令牌。在乐乐感知学堂公众号内分别回复“gpt地址”和“gpt令牌”即可获取。修改完成提交即可。
  （2）点击“启用”。这步完成后即可完成chatgpt接入公众号。
  ![image](https://user-images.githubusercontent.com/27985664/209424531-7bd06013-62e8-4190-807c-aa447efb93b1.png)
